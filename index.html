🧠 Hindi QA Hallucination Detection

IEEE Submission
PythonDeBERTa NLIXGBoostGroq APILLaMA 3.3-70B

Overview: Research paper targeting IEEE ICCUBEA/ICRITO. A hybrid pipeline for detecting hallucinations in LLM outputs for Hindi QA tasks — addressing a critical gap in low-resource NLP.

Pipeline: Groq API (LLaMA 3.3-70B) → Hindi QA dataset generation → Jaccard consistency scoring across multiple runs → DeBERTa NLI entailment scoring → XGBoost hybrid classifier.

Results: ~95% accuracy. Dataset: hindi_dataset_new180.csv. Format: 6-page IEEE double-column.

🗺️ Prahari — Nagpur Hotspot Detection

Deep Learning · Geospatial AI
PythonDeep LearningGeoPandasFoliumTensorFlowHeatmap Visualization

Overview: Prahari ("Guardian" in Hindi) is a deep learning-based crime and accident hotspot detection system built for Nagpur city. It identifies high-risk zones to aid law enforcement and urban planning.

How it works: Historical crime/accident data from Nagpur is preprocessed and fed into a deep learning model that learns spatial and temporal patterns — time of day, day of week, location clusters — to predict danger zones.

Output: Interactive heatmap overlaid on Nagpur's city map (Folium) color-coded by risk level — High (red), Medium (yellow), Low (green) — giving actionable insight for police patrolling and public awareness.

Impact: Real-world civic application of AI for public safety — making Nagpur's streets safer through data-driven hotspot prediction.
